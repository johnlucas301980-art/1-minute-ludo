# 1 Minute Ludo

A production-ready online multiplayer **1 Minute Ludo** game built with
Flutter, Node.js, Express, PostgreSQL, and Socket.IO.

> **Current Status:** Phase 1 (Project Foundation) ✅

------------------------------------------------------------------------

# Tech Stack

  Layer             Technology
  ----------------- -------------------
  Mobile            Flutter
  Backend           Node.js + Express
  Language          TypeScript
  Database          PostgreSQL
  Realtime          Socket.IO
  Version Control   Git + GitHub

# Quick Start

## Backend

``` bash
pnpm install
cp backend/.env.example .env
pnpm --filter @workspace/backend run dev
```

## Mobile

``` bash
cd mobile
flutter pub get
flutter run
```
